# FP16
Half-precision floating-point format

```
bit15:         1 bit SIGN      +---+-------+--------------+
bit14-10:      5 bit EXP       | S | EEEEE | MM MMMM MMMM |
bit0-9:       10 bit MAN       +---+-------+--------------+
```
reference [here](https://en.wikipedia.org/wiki/Half-precision_floating-point_format)
